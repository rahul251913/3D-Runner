# 3D Endless Runner Game

A simple 3D Endless Runner game developed using Unity.

## Features

- Endless running gameplay
- Obstacle spawning
- Score system
- Camera follow
- Player movement (Left, Right, Jump)

## Technologies Used

- Unity
- C#
- Universal Render Pipeline (URP)

## Project Structure

- Assets/
- Packages/
- ProjectSettings/

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/rahul251913/3D-Runner.git
   ```

2. Open the project in Unity Hub.

3. Open the main scene.

4. Press **Play** to run the game.
